魔术师(The Magician) : 创造力、能量、意愿力、行动力。
女祭司(The High Priestess) : 直觉、神秘、内在智慧、灵性的深度。
女皇(The Empress) : 丰饶、母性、创造力、与自然的连接。
皇帝(The Emperor) : 权威、稳定、领导力、秩序的力量。
教皇(The Hierophant) : 传统、信仰、道德准则、灵性导师。
恋人(The Lovers) : 爱情、关系、选择、内心的平衡。
极意(The Chariot) : 意志力、决心、胜利、掌控力。
力量(Strength) : 勇气、耐心、内在力量、控制欲望。
隐者(The Hermit) : 寻求真理、内省、孤独、灵性的指导。
命运之轮(Wheel of Fortune) : 命运的转折、循环、机遇与命运的变化。
正义(Justice) : 公正、平衡、道德、法律的力量。
吊人(The Hanged Man) : 牺牲、放弃、观察、新的觉醒。
死神(Death) : 结束、转变、重生、改变的必要性。
节制(Temperance) : 平衡、和谐、内心的调和、自我控制。
魔鬼(The Devil) : 欲望、束缚、诱惑、依赖关系。
塔(The Tower) : 剧烈的变化、解体、觉醒的痛苦、摧毁与重建。
星星(The Star) : 希望、灵感、信仰、恢复与自我发现。
月亮(The Moon) : 潜意识、幻觉、迷惑、直觉与情感的深度。
太阳(The Sun) : 光明、快乐、自信、成功与新生活的开始。
审判(Judgment) : 觉醒、重生、内心的召唤、决策与改变。
世界(The World) : 完成、圆满、整体性、融入宇宙的意义。
愚者(The Fool) : 新的开始、冒险、信任、无限的潜力。
